# Трансфузійна терапія – авторська методика (2026)

**Калькулятор для пацієнтів з ампутаціями кінцівок**

## 📚 Про метод

Калькулятор базується на трьох авторських методиках, розроблених у межах проєкту PATRIOT:

1. **Антропометрична корекція** – авторська модифікація Osterkamp (2026) з розподілом на третини.
2. **ОЦК (PATRIOT Blood Volume Adjustment, 2026)** – авторська модель корекції об'єму циркулюючої крові з використанням скоригованої маси тіла.
3. **ППТ (PATRIOT Surface Area Adjustment, 2026)** – авторська методика корекції площі поверхні тіла на основі Osterkamp + правило дев'яток.
